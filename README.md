## Build Instructions
Each exercise contains its own Makefile. To compile an exercise:

1. Navigate to the exercise directory
2. Run make
```bash
cd CPP_XX/exYY
make
```

## Usage

Each exercise creates an executable that can be run from the command line. Refer to the individual exercise directories for specific usage instructions.

## Requirements
- C++ compiler (supporting C++98)
